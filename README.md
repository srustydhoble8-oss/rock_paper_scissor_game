# 🎮 Rock Paper Scissors Game

A fun and beginner-friendly Rock Paper Scissors game built with Python! 🐍

## ✨ Features

* 🎯 User vs Computer gameplay
* 🎲 Random computer choice
* 🏆 Win, Lose, or Draw results
* 🧠 Simple game logic
* 🚀 Beginner-friendly project

## 📚 Concepts Used

* 🐍 Python Basics
* 📋 Lists
* ⌨️ User Input
* 🔀 If-Elif-Else Statements
* 🎲 Random Module
* ⚡ Logical Operators (`and`, `or`)

## 🎮 How It Works

1. 👤 The user chooses:

   * 🪨 Rock
   * 📄 Paper
   * ✂️ Scissor

2. 💻 The computer randomly selects one option.

3. ⚔️ The choices are compared.

4. 🏆 The result is displayed:

   * 🎉 You Win!
   * 🤝 Draw!
   * 😢 Computer Wins!

## 📜 Game Rules

* 🪨 Rock beats ✂️ Scissor
* 📄 Paper beats 🪨 Rock
* ✂️ Scissor beats 📄 Paper

## 🖥️ Example

Enter your choice: rock

Computer chose: scissor

🎉 You Win!
