# Rock Paper Scissors Game 🎮

A simple Rock Paper Scissors game built with Python.

## Features

* User vs Computer gameplay
* Random computer choice
* Win, Lose, or Draw results
* Easy-to-understand game logic
* Beginner-friendly project

## Concepts Used

* Python Basics
* Lists
* User Input
* If-Elif-Else
* Random Module
* Logical Operators (and, or)

## How It Works

1. The user chooses:

   * Rock
   * Paper
   * Scissor

2. The computer randomly selects one of the three choices.

3. The game compares both choices and determines:

   * Win
   * Lose
   * Draw

## Game Rules

* Rock beats Scissor
* Paper beats Rock
* Scissor beats Paper

## Example

Enter your choice: rock

Computer chose: scissor

You Win!
